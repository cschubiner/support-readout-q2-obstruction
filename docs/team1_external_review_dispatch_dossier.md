- `known theorem`: name the theorem/source and the row it covers.

- `missing hypothesis`: name the positive condition needed.
- `too broad`: name the physical admissibility axiom that removes the
  comparison class.
- `proof/package issue`: point to the exact file and line or command.
- `plausibly useful`: say which row is not obviously standard and why.

actions. Do not mutate frozen manuscripts, archives, Lean cores, or hashes
before the response is recorded.
