Purpose: sharpen the support/readout accountability premise from a bare rank
condition to a quantitative finite-precision criterion. The target is not a new
theorem in statistics or inverse problems. The target is the black-hole/AQFT
translation:

> A finite central rank or `q=2` claim from noisy operational data needs a
> condition number / deficiency / error-budget statement, not only a formal
> separating family of probes.

## Setup

Let

```text
Z_F = l^\infty(F),       V_F = Z_F / C1,       dim V_F = |F|-1.
```

Equip `V_F` and the readout space `W` with fixed norms. Let

```text
R : V_F -> W
```

be the linear map generated by the declared central probes, effects, sector
records, recovery task, spectroscopy channel, or other finite-center readout.

The exact Team 1 accountability condition is `ker R = 0`, or equivalently
`rank R = dim V_F` after quotienting declared null/gauge/inadmissible
directions. In finite precision, this is not enough. One also needs a lower
inverse-modulus bound

```text
alpha(R) = inf_{||v||=1} ||R v|| > 0
```

and an error budget.

## Proposition: Exact And Noisy Accountability

1. **Exact criterion.** `R` separates all finite central contrasts if and only
   if `alpha(R)>0`; in finite dimension this is equivalent to `ker R=0`.

2. **Stable reconstruction.** If observed readout data satisfy

   ```text
   ||R v - y|| <= epsilon,
   ```

   then every exact compatible contrast obeys

   ```text
   ||v - v'|| <= 2 epsilon / alpha(R)
   ```

   for any two contrasts `v,v'` compatible with the same noisy data ball.
   Conversely, if two physical contrasts differ by more than the required
   resolution `delta`, then they are distinguishable under error `epsilon`
   only when

   ```text
   alpha(R) delta > 2 epsilon.
   ```

3. **Instability/hidden-direction criterion.** If `alpha(R)=0`, there is a
   nonzero central contrast invisible to the readout. If `alpha(R)` is positive
   but very small, the readout is formally separating but practically unable to
   support a rank or binary claim at the stated error budget.

4. **Null/gauge quotient form.** If a null/gauge/inadmissible subspace `N` is
   declared, the same statements hold for the induced map

   ```text
   R_bar : V_F / N -> W
   ```

   with `alpha(R_bar)`.

## Proof

The exact criterion is finite-dimensional linear algebra. If `alpha(R)>0`,
then `||Rv||=0` implies `v=0`, so `ker R=0`. If `ker R=0`, compactness of the
unit sphere gives a positive minimum for `||Rv||`.

For noisy data, suppose both `v` and `v'` are compatible with the same observed
readout `y`. Then

```text
||R(v-v')|| <= ||Rv-y|| + ||Rv'-y|| <= 2 epsilon.
```

By the lower inverse-modulus bound,

```text
alpha(R) ||v-v'|| <= ||R(v-v')||,
```

so `||v-v'|| <= 2 epsilon/alpha(R)`. The distinguishability condition is the
contrapositive: to separate contrasts at distance at least `delta`, the noisy
compatibility radius must be smaller than that distance.

The quotient statement applies the same argument to `V_F/N`.

## Consequence For Binary Rank

A binary conclusion `q=2` from noisy central readouts requires all of:

1. a declared physical contrast space, after null/gauge/inadmissible quotient;
2. full exact rank on that contrast space;
3. a positive lower inverse-modulus `alpha` large enough for the experimental
   or theoretical error budget;
4. a finite-alphabet/maximality rule saying the accounted one-dimensional
   contrast is the whole physical finite center, not a quotient of a larger
   hidden center;
5. or direct `q=2` spectroscopy with a stated line-resolution/error budget.

Without item 3, formal separation is too ill-conditioned to justify a
finite-rank claim. Without item 4 or 5, a well-conditioned binary readout may
still be only a binary quotient of a nonbinary center.

## Referee Classification

Classify this note as one of:

- `known`: this is standard finite-dimensional inverse stability plus
  Blackwell/Le Cam/Petz/Buscemi sufficiency/deficiency; cite the exact theorem
  and demote Team 1 to an application note.
- `false`: a standard AQFT/OAQEC/spectroscopy setting supplies a different
  stability principle that avoids this error-budget requirement; name it.
- `missing hypothesis`: the target physics already assumes a lower readout
  modulus, full central tomography, or a line-resolution budget, but the
  frozen package did not state it.
- `plausibly useful`: the theorem is elementary, but the explicit
  finite-center warning is useful for black-hole/AQFT reconstruction claims.

## Consequence For Team 1

Team 1 should not say that a formally separating readout has produced a
referee-grade finite-rank or `q=2` result until it also states the quantitative
readout modulus and the error budget. This is especially important for
spectroscopy and finite operational probes: a line or central contrast only
counts as a rank-breaking datum when the separation margin exceeds the stated
uncertainty.
